How to Use

1. Mint an NFT:

To create a new NFT, interact with the `Minty` contract's `mintToken` function. You need to provide the token owner's address and the metadata URI for the NFT.

2. List an NFT for Auction:

Use the `list` function in the `Auction` contract to list an NFT for auction. You need the NFT contract address, token ID, minimum price, and auction duration in hours.

3. Place a Bid:

To bid on an auction, call the `bid` function of the `Auction` contract with the listing ID and your bid amount. Ensure your bid exceeds the current highest bid and the minimum price.

4. End an Auction:

Once the auction ends, use the `end` function in the `Auction` contract to finalize the auction. This transfers the NFT to the highest bidder and sends the auction proceeds to the seller.

5. Withdraw Funds:

If you have been outbid, or if you have funds from an auction, use the `withdrawFunds` function to withdraw your balance.
